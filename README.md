# 云原生示例

### 内容
- [ ] Kubernets 自动部署、扩展和管理容器化应用程序
- [ ] CoreDNS 服务发现插件
- [ ] Calico网络插件
- [ ] Envoy 云原生高性能边缘/中间/服务代理
- [ ] Helm 包管理
- [ ] Prometheus 监控系统和时间序列数据库
